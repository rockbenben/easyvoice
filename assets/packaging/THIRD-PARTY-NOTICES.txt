易声(EasyVoice) 整合包 —— 第三方组件许可声明
=================================================

为实现“解压即用”，本整合包内置了 Python 运行时、Qwen3-TTS 模型权重与 FFmpeg。
各第三方组件的许可与归属如下。易声(EasyVoice) 自身代码的许可见项目仓库根目录的
LICENSE 文件。

1) Qwen3-TTS 模型权重  ——  models\Qwen\Qwen3-TTS-12Hz-0___6B-Base\
   许可：Apache License 2.0（允许再分发与商用）。
   完整许可文本见该模型目录内的 LICENSE 文件，以及
   https://www.apache.org/licenses/LICENSE-2.0
   版权归 Qwen 团队 / 阿里巴巴所有；项目主页 https://github.com/QwenLM/Qwen3-TTS
   易声在其基础上仅做应用层封装，未修改模型权重。

2) FFmpeg  ——  ffmpeg\ffmpeg.exe
   许可：GNU General Public License v3 (GPLv3)，gyan.dev 的 “release-essentials” 构建。
   完整许可文本与源代码获取声明见 ffmpeg\COPYING.GPLv3.txt 与 ffmpeg\README-LICENSE.txt。
   易声以独立子进程方式调用，属“单纯聚合”，GPLv3 不传染到易声自身代码。

3) 内置 Python 运行时及其依赖  ——  runtime\
   通过 conda-pack 打包，内含 Python 3.12、PyTorch、Gradio、qwen-tts 等及其全部依赖。
   这些第三方库各自的许可证（多为 BSD / MIT / Apache-2.0 / PSF 等宽松许可）随包附带，
   位于各自的发行信息目录：
       runtime\Lib\site-packages\<包名>-<版本>.dist-info\  （内含 LICENSE / METADATA）
   PyTorch 的 CUDA 运行库为 NVIDIA 可再分发组件，依其各自条款随附。
   如需某具体组件的完整许可文本，请查阅对应的 .dist-info 目录。

如对任一组件的许可或源代码获取有疑问，请通过项目仓库 issue 联系维护者。
